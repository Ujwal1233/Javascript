Event Capturing *The capturing phase is the first phase of event propagation,in the DOM *Event capturing means the event is triggered first of an outermost or parent element and then it goes to the innermost or child element. *To perform event capturing for the add event lisyener along with the event type and the callback function we have to pass a third argument as true syntax: *element.addEventListener(event, callback, true) // true for event capture *element.addEventListener(event, callback, false) // false for event bubbling *element.addEventListener(event, callback) // by default it is false for event bubbling Document
